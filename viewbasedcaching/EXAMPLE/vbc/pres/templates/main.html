View based caching

This is a sample implementation of the view based caching concept. Please visit the APF documentation page to get the details on the implementation. The following box defines a view, that is not cached, the second box includes the content of a cached view using the <cache:importdesign /> tag's functionality:

Usage:

  • Press F5 to see, that the timestamp in the second view is not refreshed.
  • Add clearcache=true to the url to clear the cache.
  • Add benchmarkreport=true to the url to see the benchmark report.

Note:

  • The caching is done by the filesystem cache provider.
  • The folder, where the index.php resides must be writable for the webserver user.
