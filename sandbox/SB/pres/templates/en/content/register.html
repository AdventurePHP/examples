Registration

On this page you can sign-up as a new user. This component is designed for your customers to create a new user account. The present implementation is a simple form that requires only the basic data of a user to be filled in. In case you want to store further attributes this implementation can be taken as a good start.

Please note, that you can only sign up in case the user management wizzard has been executed before.

In order to include the form, please use the subsequent code: 

<core:importdesign
   namespace="APF\modules\usermanagement\pres\templates\registration"
   template="register"
/>

The appearance of the form can be defined using CSS freely. Details can be taken from the /css/register.css and this template that is located under /APF/sandbox/pres/templates/en/content/register.html.
