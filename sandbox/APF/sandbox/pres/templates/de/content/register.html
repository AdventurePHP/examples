Registrierung

Auf dieser Seite können sie sich als neuer Benutzer registrieren. Diese Komponente des Usermanagement-Moduls ist für ihre Kunden gedacht, um sich an ihrer Applikation anzumelden. Es handelt sich dabei jedoch nur um ein einfaches Formular, das die Basis-Daten eines Benutzers abfragt, die auch im Modul gespeichert werden. Sollen weitere Attribute abgefragt werden, kann dieses jedoch als Basis für eine eigene Implementierung genutzt werden.

Bitte beachten Sie, dass eine Registrierung nur möglich ist, wenn sie den Usermanagement-Wizzard bereits ausgeführt haben.

Um das Formular einzubinden, ist folgender Code notwendig: 

<core:importdesign
   namespace="APF\modules\usermanagement\pres\templates\registration"
   template="register"
/>

Die Darstellung ist per CSS frei definierbar. Details hierzu können den Dateien /css/register.css und dem Template unter /APF/sandbox/pres/templates/de/content/register.html entnommen werden.
