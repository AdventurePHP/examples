APF modules demo

This is the main template of the application, that can include a dynamic modules. Each module is defined by a url key that refers to a module definition located in the database.

A module can have a navi view and a content view, that is specified within the database table together with a front controller based action. The action class can be used to initialize the module before the view layer is initialized.

In order to call another module, please add the mod url parameter containing the key of the desired module to call.
